# 🤖 UnoBot — Игровая платформа для обучения Python

Полноценный Telegram-бот с геймификацией и обучением.

### 🌟 Основные фичи:
* **Турниры:** Еженедельные битвы за XP и места в топе.
* **Геймификация:** Профили, уровни, достижения (14 штук).
* **Магазин:** Покупка Premium и бонусов за внутреннюю валюту.
* **Рефералка:** Система приглашений для вирального роста.
* **Обучение:** Квизы, мини-игры "Найди баг" и курсы.
* **Мультиязычность:** Полная поддержка RU/EN.
